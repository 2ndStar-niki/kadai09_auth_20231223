# 課題　 PHPでログイン機能を実装する

## ①課題内容　
- ユーザー登録機能とログイン機能が実装されているECサイトを作成した。管理側のアプリで商品情報の追加、修正、削除を可能にし、顧客側のアプリで表示されている商品の購入を可能にした。

## ②工夫した点・こだわった点
- ユーザー登録機能の実装
- ログイン中ユーザー名の表示

## ③難しかった点・次回トライしたいこと(又は機能)
- お気に入り登録機能
- 購入履歴機能
- 購入完了メール送付機能

## ④質問・疑問・感想、シェアしたいこと等なんでも
- [感想]データベースに接続して商品情報とユーザー情報を取得しているが、それぞれの接続関係がいまいち理解できていないので、改めて動画学習に励みたい。また、ネットの記事を参考にお気に入り登録機能を実装しようとしたが、いまいち実装したい機能とのイメージが合わず、お気に入り登録機能の実装ができていない。このままでは、自分が実装したい機能に対してどこか妥協しないとプロダクトを完成させることができないので、0からコードを書けなくても必須で実装したい機能に対しては、何とかして実装できるようにしたい。
